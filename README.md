🍽️ Swiggy Data Analysis & Data Modeling using SQL

📌 Project Summary

Designed and implemented a SQL-based data analysis and data modeling solution on Swiggy food delivery data. The project focuses on data cleaning, transformation, and schema design, enabling efficient querying and analytical insights.

This project demonstrates hands-on experience with data engineering fundamentals, including data validation, deduplication, and dimensional modeling.

🎯 Key Highlights

Cleaned and transformed raw data using SQL (CTEs, Window Functions)

- Identified and removed duplicate records using ROW_NUMBER()
- Performed data quality checks (NULL handling, blank values)
- Designed dimension tables for structured analytics
- Applied data modeling concepts (star schema basics)
- Improved dataset usability for analytical querying and reporting

🛠️ Tech Stack
- SQL (SQL Server)
- Data Cleaning & Transformation
- Data Modeling (Dimensional Modeling)
